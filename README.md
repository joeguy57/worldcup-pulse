# WorldCupPulse: Did 2026 Outperform 2022?
> Comparing stadium attendance and global TV viewership between
> the 2022 Qatar World Cup and the 2026 USA/Canada/Mexico edition.
## Stack
- Python — data ingestion from API-Football
- Apache Spark (PySpark) on Databricks — bronze to silver transformation
- dbt — mart layer modeling and data quality tests
- Tableau — comparative dashboard
## Key Question
Did the expanded 48-team format and North American timezones
produce higher attendance and viewership than 2022?
